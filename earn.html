📚 Understanding the Tokens

Two forms of DASH that power the ecosystem

💎
What is DASH?
DASH (~$35) is the native cryptocurrency that powers Dash Platform — a decentralized data storage layer with 200+ EvoNodes worldwide. It's been running since 2014 and is battle-tested for reliability. Platform fees are paid in DASH, making it the fuel for permanent agent memory.
🔗
What is wDASH?
wDASH (Wrapped DASH) is an ERC-20 token on Pentagon Chain that represents real DASH. You can burn wDASH to redeem real DASH on Dash L1 at any time. It's backed by 4,000 DASH in EvoNode collateral — not a synthetic or algorithmic peg, but real locked value. Use wDASH for swaps, LP, and agent funding on-chain.
